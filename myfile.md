# Learn Git Bash with niBVL
## Getting Started
Create file:

    mkdir <file name>

Iniial git: 

    git init

Check change git :

    git status

Add change:

    git add <file name> 
    git add .

Commit:

    git commit -m "<message>"

Create branch:

    git branch <branch name>

Switch branch:

    git checkout <branch name>

Create and switch branch:

    git checkout -b <branch name>

Merge branch:

    git merge <branch name>
